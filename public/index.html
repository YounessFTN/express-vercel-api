API Documentation

Route Users

POST /users

Create a new user

Body: { "name": "string", "email": "string", "password": "string" }

GET /users

Get the list of all users

GET /users/:id

Get a specific user by ID

URL params: id (required)

PUT /users/:id

Update a user

URL params: id (required)

Body: { "name": "string", "email": "string", "password": "string" }

DELETE /users/:id

Delete a user

URL params: id (required)
